# Welcome 

Branche "propre" pour le développement de l'UI

<h2>Architecture finale (idée en cours):</h2>
-ui.py: fichier contenant le code de l'UI de QtDesigner <br>
-gui_function.py: fichier rassemblant toutes les fonctions pour manipuler l'interface graphique <br>
-LogReaderThread.py: Lecture et sortie du chat/log <br>
-server.py: Manipulation du serveur PHP <br>
-main.py: fichier main du projet <br>
-image: dossier contenant l'ensemble des images utilisées pour l'UI <br>
-(chat.py: fichier rassemblant les fonctions pour manipuler et intéragir avec le chat et les logs)? <br>

<h3>Dépendance:</h3>
-pyside6: pip install PySide6 <br>
-librairie custom de KhamisiKibet: pip install QT-PyQt-PySide-Custom-Widgets <br>

<h3>Instruction</h3>
-Lancé la commande "make run" <br>
