# Dungeon Kitchen

An application to play TTRPG online. The application provides the tools for the GameMaster to create objects, characters and maps. The GameMaster can open a local server to start the game. An URL is generated at the start of the server. The players can then connect to the game by entering the URL given by the GameMaster.

The application is composed of 3 parts:
    -The editor that allows for the creation of the objects for the g(items, abilities, character, map).
    -The gamemode that allows the GameMaster to play the game in a special interface with special interactions
    -The site where the players can see the map, their character sheet and roll dices

# Instruction

To launch the application, type "make run" in a terminal in the Dungeons-Kitchen's directory
